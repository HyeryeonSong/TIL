# Git - Git branch 관리 🧮



## **branch 생성**
```
$ git branch test master
```
: master 브랜치에 test 라는 브랜치 생성

<br>

## **branch 삭제**
```
$ git branch -D test
```
: test 라는 브랜치 삭제

<br>

## **branch 이름 변경**
```
$ git branch -m test testcode
```

: test 라는 브랜치를 testcode 로 이름 변경

<br>

## **현재 브랜치 확인**

```
$ git branch
```
: 현재 브랜치 확인   
: 명령어 입력 후 나올땐 :q 를 입력하면 된다.
